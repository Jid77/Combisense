Combisense — Quick User Guide ( Utility Engineering)

Monitoring & kontrol utility • Real-time status • Alarm • Grafik • Ekspor Excel

Quick Start

  • Install APK, buka & izinkan Notifikasi + Penyimpanan.
  • Pastikan HP terhubung internet.
  • Pertama kali, data grafik terisi bertahap (interval penarikan data ±10 menit).
    • Selain data untuk grafik, Data akan ditarik secara real-time
  • Setelah dibuka sekali, app berjalan di background.
    Jika HP restart/mati, buka app lagi agar penarikan data aktif.

Navigasi Halaman

  • Indikator: ringkasan status Normal/Abnormal.
  • Artesis: PV/SV, set preset, reset, ON manual by timer.
  • M800: jam UV Lamp + TOC/Temp/Conduct.
  • Vent Filter: PV/SV Tk201/Tk202/Tk103 + grafik.
  • LBENG AHU 04: PV/SV TF3, Temp, RH + grafik.

Artesis (tips)

  • Setting SV: Tombol “Setting” → verifikasi PIN.
  • Reset timer: Tombol "Reset" → verifikasi PIN
  • ON Pompa manual by timer: Toggle "on/off" → verifikasi PIN pompa akan on sesuai pilihan timer lalu otomatis OFF mencegah human error

Vent Filter (tips)

  • Setting SV: Tombol “Setting” → verifikasi PIN.
  • Range ideal: 65–80 °C (area hijau di grafik).
  • Tooltip: sentuh & geser grafik untuk melihat nilai & jam.
  • Dropdown “X pts” mengubah jendela data; legend bisa di-toggle.

M800 (tips)

  • UV Lamp: bar warna Hijau → Kuning → Merah saat mendekati batas.
  • TOC/Temp/Conduct: grafik mini; sumbu otomatis; label jam.

LBENG AHU 04 (tips)

  • Setting SV: Tombol “Setting” → verifikasi PIN.
  • Range umum: Temp 18–27 °C, RH 45–80%.
  • Tooltip & jendela data sama seperti Vent Filter.

Alarm & Notifikasi

  • Boiler, OFDA, Chiller, Vent Filter, AHU 04, dll (sesuai pengaturan).
  • Saat terjadi alarm: notifikasi lokal “Sensor Alarm” berbunyi.
  • Histori alarm dapat dilihat di halaman histori
  • ON/OFF alarm dapat diatur di menu Settings.

Ekspor ke Excel

  • Pilih tanggal mulai–akhir, izinkan penyimpanan.
  • File tersimpan di Download/combiphar/exported_data.xlsx
  • Sheet tersedia: Sensor Data & Alarm History.

Troubleshooting

  • Blank/Hitam: cek internet, paksa tutup (force close) & buka lagi.
  • Grafik tidak bergerak: lakukan pull-to-refresh; pastikan data sensor berjalan.
  • Notifikasi tidak muncul: izinkan notifikasi & pastikan mode Jangan Ganggu mati.
  • HP restart: buka app lagi agar penarikan data aktif.

Catatan Teknis

  • Data dibaca dari Firebase Realtime Database (sensor_data / commands).
  • Grafik mengoleksi data berkala (±10 menit) dan menyimpan cache lokal di smartphone masing-masing.
  • Saat mengatur SV atau reset memerlukan waktu beberapa detik sampai value atau tampilan di aplikasi berubah (dikarenakan menunggu write ke perangkat asli selesai)
